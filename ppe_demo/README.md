# PPE云端智能大礼包 - 核心流程Demo

## 📋 Demo说明

本Demo演示PPE云端智能大礼包的**核心流程**：
1. 资料上传与归档
2. 心得体会收集
3. AI生成课程文档

## 🚀 快速开始

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 运行Demo
```bash
python main.py
```

### 3. 查看输出
生成的课程文档保存在 `output/course_docs/` 目录下。

## 📊 Demo流程

### 步骤1: 扫描现有资料包
- 自动扫描 `PPE大二上资料包` 文件夹
- 识别9门课程、127个文件
- 自动归档并生成标准化文件名

### 步骤2: 生成示例心得体会
- 生成4条示例心得（中国经济概论、世界经济概论、西方政治思想史、概率论）
- 调用智谱AI提取关键信息
- 自动审核通过

### 步骤3: 生成课程文档
- 为每门课程生成完整文档（6个部分）
- 基于心得体会和资料列表
- 输出Markdown和JSON格式

## 🎯 核心功能

### 1. 资料上传
- 标准化文件名：`{资料类型}_{贡献者}_{原始文件名}`
- 自动识别资料类型（PPT/笔记/真题等）
- 记录元信息（贡献者、课程、推荐理由）

### 2. 心得收集
- 结构化提交（作者、年级、课程、成绩、内容）
- AI自动提取关键信息（课程概述、难点、老师偏好）
- 审核流程（学委验证成绩真实性）

### 3. 文档生成
- 固定6部分模板
- 基于高分学长学姐心得
- 资料串讲（说明如何使用资料）
- 贡献者列表（增强参与感）

## 🔧 技术栈

- **Python 3.9+**
- **智谱AI GLM-4-Flash** - 文档生成
- **asyncio** - 异步处理
- **httpx** - HTTP客户端

## 📌 注意事项

- 本Demo使用本地文件模拟上传，实际部署需对接飞书云空间
- 心得体会的示例内容由AI生成，实际使用需真实学长学姐撰写
- 智谱API Key已配置在 `config.py` 中
