# PPE云端智能大礼包 - 知识索引

## 背景知识

### 南开PPE专业
- **PPE**：哲学、政治学、经济学（Philosophy, Politics and Economics）
- 南开大学PPE实验班跨学科培养项目
- 每届约30人，学习委员负责资料传承

### 现有大礼包机制
- 以学期为单位的压缩包
- 学习委员手工整理、更新、传递
- 包含：PPT、笔记、真题、阅读材料等

---

## 技术知识

### 飞书知识库
- **多维表格**：类似Notion数据库，可创建课程列表
- **飞书文档**：课程文档载体，支持内嵌资料
- **权限体系**：可控制不同角色的读写权限

### AI Agent技术栈（待确定）
| 层面 | 可能选项 |
|------|----------|
| 知识库 | 飞书多维表格 + 飞书文档 |
| AI Agent | Python + LangChain + 飞书API |
| 大模型 | 飞书智能伙伴 / 外接API |
| 文件存储 | 飞书云盘 |
