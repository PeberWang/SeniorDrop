# 00 · 快速开始（5 分钟跑通）

> 目标读者：第一次部署的管理员。前置：已装 Python 3.11+、LibreOffice。
>
> 跑完本文档后，你会有一个能用的飞书知识库骨架（4 个学年文档 + 课程导航 + 占位课程文档），但没有真实资料。真实资料通过表单收集 → 详见 [01-管理员指南.md](01-管理员指南.md)。

---

## 一、准备账号

### 1.1 飞书企业版

1. 注册飞书企业版（免费版 50 人内即可）：https://www.feishu.cn/
2. 进入 https://open.feishu.cn/ → 创建企业自建应用
3. 应用开通以下权限（权限管理 → 申请权限）：
   - `wiki:wiki`（知识库读写）
   - `docx:document`（文档读写）
   - `bitable:app`（多维表格读写）
   - `drive:drive`（云盘读写）
   - `drive:media:upload`（媒体上传）
4. 拿到 App ID 和 App Secret

### 1.2 阿里云 OSS

1. 注册阿里云账号：https://www.aliyun.com/
2. 开通对象存储 OSS：https://www.aliyun.com/product/oss
3. 新建一个 bucket：
   - 名称：自定义（如 `xx-major-giftbox`）
   - 区域：选最近的（如华东1-杭州 / 华北2-北京）
   - **ACL**：选「公共读」（学生需要长期可访问下载链接）
4. 拿到 AccessKey ID + AccessKey Secret（建议 RAM 子账号，最小权限）

### 1.3 DeepSeek API

1. 注册 DeepSeek：https://www.deepseek.com/
2. 充值（最少 10 元能用很久）
3. 拿到 API Key

### 1.4 智谱 GLM-OCR

1. 注册智谱开放平台：https://open.bigmodel.cn/
2. 充值（OCR 比较便宜，10 元能 OCR 几十份 PDF）
3. 拿到 API Key

## 二、安装

```bash
# 1. 下载项目代码
git clone <your-repo-url>
cd 课程资料智能大礼包

# 2. 安装 Python 依赖
pip install -r requirements.txt

# 3. 装 LibreOffice（Office 文档转 PDF 用，OCR 流程必需）
# Windows：https://www.libreoffice.org/dl/win-x86_64/
# macOS：brew install libreoffice
# Linux：apt install libreoffice
```

## 三、配置 .env

```bash
cp .env.example .env
```

打开 `.env`，按下面填入（字段含义详见 [04-技术原理.md](04-技术原理.md)）：

```bash
# === 飞书（1.1 拿到）===
FEISHU_APP_ID=cli_xxxxxxxxxxxxxxxx
FEISHU_APP_SECRET=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
FEISHU_BASE_URL=https://open.feishu.cn/open-apis
FEISHU_DOC_HOST=feishu.cn

# === LLM (DeepSeek，1.3 拿到)===
LLM_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
LLM_BASE_URL=https://api.deepseek.com
LLM_MODEL=deepseek-v4-pro

# === OCR (智谱，1.4 拿到)===
GLM_API_KEY=xxxxxxxxxxxxxxxx.xxxxxxxxxxxxxxxx
GLM_OCR_URL=https://open.bigmodel.cn/api/paas/v4/layout_parsing

# === 阿里云 OSS（1.2 拿到）===
CLOUD_DRIVE_BACKEND=aliyun_oss
OSS_BUCKET=your-bucket-name
OSS_ENDPOINT=https://oss-cn-beijing.aliyuncs.com   # 按你 bucket 的区域调整
OSS_ACCESS_KEY_ID=LTAIxxxxxxxxxxxxxxxx
OSS_ACCESS_KEY_SECRET=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
OSS_PRESIGNED_TTL=86400
# 公共读直链（bucket 改公共读后启用，学生长期可下载）
OSS_PUBLIC_BASE=https://your-bucket-name.oss-cn-beijing.aliyuncs.com

# === 路径配置 ===
MATERIALS_BASE=./data/courses/大二上

# === 知识库 + bitable ===
WIKI_SPACE_NAME=XX 专业课程大礼包
# BITABLE_APP_TOKEN 先留空，首次跑 init-bitable 后填回

# === 日志 ===
LOG_LEVEL=INFO
LOG_TO_FILE=true
LOG_FILE_PATH=./logs/app.log
```

## 四、首次部署（5 命令）

按顺序执行：

```bash
# 1. 初始化 bitable（建三张管理表）
python deploy.py init-bitable
```

返回值会有 `app_token`，把它复制到 `.env` 的 `BITABLE_APP_TOKEN=`。

```bash
# 2. 设置 bitable 链接分享（凭链接即可访问，方便管理员 UI 操作）
python deploy.py open-bitable

# 3. 录入课程清单
```

准备一个 Excel/CSV 文件，列名：`名称 / 学期 / 授课老师 / 考核方式`（`类型` 列可选，默认专业必修课）。示例（保存为 `data/课程清单.xlsx`）：

| 名称 | 学期 | 授课老师 | 考核方式 |
|---|---|---|---|
| 概率论与数理统计 | 大二下 | 刘会刚 | 闭卷 |
| 微观经济学 | 大一上 |  | 闭卷 |
| ... | ... | ... | ... |

```bash
# 批量导入课程
python deploy.py seed-course --from-file data/课程清单.xlsx

# 单条录入也可以
python deploy.py seed-course --name 概率论与数理统计 --semester 大二下 --teacher 刘会刚 --type 专业必修课 --exam 闭卷
```

```bash
# 4. 建飞书知识库 + 4 个学年文档 + 内嵌课程导航表
python deploy.py wiki

# 5. 生成每门课的学习指南文档（首次跑只有基本信息，串讲段落是占位）
python deploy.py docs

# 6. 回填导航表的「学习指南」字段（让链接可点）
python deploy.py link
```

完成后，去飞书知识库，应能看到 4 个学年文档，每个文档里有这门学年的课程导航表，每行有可点击的「学习指南」链接。

## 五、验证

打开飞书知识库（在飞书云文档 → 知识库），找到你刚建的 `XX 专业课程大礼包` 知识空间：

1. 4 个文档（大一 / 大二 / 大三 / 大四）存在
2. 每个文档顶部有标题 + 简介 + 课程导航表
3. 导航表里每行有课程名 + 老师 + 学期 + 类型 + 考试 + 「学习指南」链接
4. 点「学习指南」链接，跳到该课的独立文档，里面有：
   - 授课老师 / 学期 / 考试 信息
   - 课程概述（LLM 生成，可能写「待同学补充」）
   - 推荐资料表（首次为空）
   - 资料串讲（首次是占位）
   - 贡献者表（首次为空）

如果以上都正常，**首次部署成功**。

## 附：完整命令清单

```bash
# 初始化 / 配置
python deploy.py init-bitable        # 建管理 bitable 三张表
python deploy.py open-bitable        # 设置链接分享
python deploy.py grant-bitable       # 加协作者（指定邮箱 / openid）

# 录入数据
python deploy.py reset-bitable       # 清空三张表所有记录（危险！）
python deploy.py seed-course         # 录课程（单条 / --from-file 批量）
python deploy.py seed-materials      # 录资料（本地文件夹 → 飞书 drive + 资料表）

# 资料处理链路
python deploy.py archive-materials   # 飞书附件 → OSS
python deploy.py ocr-materials       # OCR + 摘要（OSS → PDF → OCR → 摘要 → 飞书 drive + 回填）

# 知识库生成
python deploy.py sync                # 聚合 bitable + 门控
python deploy.py wiki                # 建/更知识库 + 学年文档 + 导航表
python deploy.py docs                # 生成课程文档
python deploy.py link                # 回填导航表链接

# 维护
python deploy.py fix-bitable         # 修复单选字段选项
python deploy.py purge-archived      # 清理归档超期原件
python deploy.py logs                # 查看日志

# 一键全流程
python deploy.py full                # wiki + tables + docs + upload + link
```

每条命令加 `--help` 看详细参数，如 `python deploy.py seed-course --help`。
